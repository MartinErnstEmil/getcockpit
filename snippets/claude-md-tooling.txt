---
title: Prefer dedicated tools over Bash
description: Use Glob/Grep/Read/Edit, not find/grep/cat
target: claude_md
section: tools
priority: 85
tags: tools, claude-code
---
- Use Glob tool for finding files — do NOT use Bash(find) or Bash(ls)
- Use Grep tool for searching — do NOT use Bash(grep) or Bash(rg)
- Use Read tool for reading files — do NOT use Bash(cat/head/tail)
- Use Edit tool for modifying files — do NOT use Bash(sed/awk)

- When you intend to call multiple independent tools, make all the calls in parallel within the same response
- For dependent operations, call sequentially — never use placeholders for unknown values
